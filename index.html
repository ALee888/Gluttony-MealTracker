CPSC 314 Final Project Presentation

Gluttony

Adam Lee

Functional Requirements:

  1. Users must be able to calculate macros from personal information

  2. Users must be able to see daily or weekly progress towards their goals

  3. Users must be able to create a new meal entry

  4. Users must be able to clearly see their past eating habits from a calendar view

  5. Users must be able to clearly see their future eating habits form a calendar view

  6. Users must be able to create an account and log in via username and password

  7. Users must be able to create a new meal and save it to the database

  8. Users must be able to save their nutritional goals to a database

  9. Users must be able to find nutritional information on food from a database

  10. Users must be able to see past and future meals from the database
